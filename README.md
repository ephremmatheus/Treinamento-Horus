# Treinamento PHP – Sistema de Cadastro de Pessoas

Este projeto foi desenvolvido como parte de um **treinamento técnico proposto pela Horus**, utilizado no processo seletivo da empresa para avaliação de conhecimentos em desenvolvimento com PHP.

O objetivo do treinamento é evoluir gradualmente a construção de um sistema web, aplicando conceitos fundamentais de programação, banco de dados e organização de código.

O sistema implementa um **CRUD (Create, Read, Update, Delete)** para cadastro de pessoas, sendo desenvolvido em diferentes níveis ao longo do treinamento.

O treinamento tem como base o livro **"PHP: Programando com Orientação a Objetos"**, de **Pablo Dall'Oglio**, utilizado como referência para a estrutura do projeto e para a aplicação dos conceitos de orientação a objetos em PHP.

---

## 🚀 Funcionalidades

* Cadastro de pessoas
* Listagem de registros
* Edição de dados
* Exclusão de registros
* Conexão com banco de dados
* Organização do código em classes

---

## 🧠 Conceitos praticados

Durante o desenvolvimento deste projeto foram aplicados diversos conceitos importantes do desenvolvimento web:

* Formulários HTML
* Manipulação de requisições (`GET` e `POST`)
* Conexão com banco de dados utilizando **PDO**
* Execução de **queries SQL**
* Estruturação de aplicações em múltiplos arquivos
* Separação entre **interface, lógica de aplicação e acesso a dados**

---

## 🧩 Programação Orientada a Objetos

O projeto também utiliza conceitos de **Programação Orientada a Objetos (POO)** em PHP para melhorar a organização e manutenção do código.

Entre os conceitos utilizados estão:

* **Classes** para representar entidades do sistema
* **Métodos** responsáveis pelas operações de persistência no banco de dados
* **Atributos estáticos** para controle da conexão com o banco
* **Encapsulamento**, separando responsabilidades dentro da aplicação

Um exemplo é a classe responsável pela manipulação dos dados da entidade **Pessoa**, onde são implementadas operações de persistência no banco de dados.

## ⚙️ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/ephremmatheus/Treinamento-Horus.git
```

2. Coloque o projeto dentro do diretório do servidor local
   (exemplo usando XAMPP)

```
xampp/htdocs/
```

3. Crie o banco de dados no **MySQL**

4. Configure os dados de conexão no projeto

5. Acesse no navegador:

```
http://localhost/treinamento
```
